Версия: 1.0
Язык: C++17

1. ОБЩЕЕ ОПИСАНИЕ
Данный модуль предоставляет набор функций для выполнения базовых алгебраических вычислений: извлечение корней произвольной степени, возведение чисел в степень, вычисление логарифмов по произвольному основанию и вычисление факториалов. Модуль предназначен для использования в учебных, научных и прикладных проектах, требующих точных математических операций.

2. ПРОСТРАНСТВО ИМЕН
Все функции модуля объявлены внутри пространства имен Algebra. При подключении заголовочного файла algebra.h необходимо явно указывать префикс Algebra:: перед вызовом функций либо использовать директиву using namespace Algebra; внутри исходных файлов реализации (.cpp). Использование пространства имен обязательно для предотвращения конфликтов имен с другими библиотеками и соблюдения стандартов проектирования C++.

3. СПРАВОЧНИК ФУНКЦИЙ 

3.1. nthRoot
Назначение: Вычисляет корень степени degree из числа number.
Параметры:
  - number (double): число, из которого извлекается корень.
  - degree (int): степень корня (должна быть положительным целым числом).
Возвращаемое значение: double - результат вычисления корня.
Ограничения:
  - degree должно быть строго больше 0.
  - Если number отрицательное, а degree четное, вычисление невозможно в области действительных чисел.
  - Точность результата соответствует точности типа double (около 15-17 значащих цифр).

3.2. power
Назначение: Возводит число base в степень exponent.
Параметры:
  - base (double): основание степени.
  - exponent (double): показатель степени.
Возвращаемое значение: double - результат возведения в степень.
Ограничения:
  - Функция использует стандартную реализацию std::pow.
  - Поведение при base = 0 и exponent <= 0, а также при отрицательном base и дробном exponent зависит от реализации стандартной библиотеки C++. Рекомендуется проверять входные данные в вызывающем коде.

3.3. logarithm
Назначение: Вычисляет логарифм числа number по основанию base.
Параметры:
  - number (double): аргумент логарифма.
  - base (double): основание логарифма.
Возвращаемое значение: double - значение логарифма.
Ограничения:
  - number должно быть строго больше 0.
  - base должно быть строго больше 0 и не равно 1.
  - Вычисление выполняется по формуле перехода: log_base(number) = ln(number) / ln(base).

3.4. factorial
Назначение: Вычисляет факториал целого неотрицательного числа n.
Параметры:
  - n (int): число, факториал которого требуется вычислить.
Возвращаемое значение: unsigned long long - точное значение факториала.
Ограничения:
  - n должно быть в диапазоне от 0 до 20 включительно.
  - Значение 20! является максимальным, которое помещается в тип unsigned long long без переполнения.
  - 0! возвращает 1.
4. ОБРАБОТКА ОШИБОК
Модуль использует стандартные исключения C++ для сигнализации о некорректных входных данных или невозможности вычисления. Все исключения наследуются от std::exception.

4.1. std::invalid_argument
Выбрасывается в следующих случаях:
  - nthRoot: параметр degree <= 0.
  - logarithm: параметр base <= 0 или base == 1.
  - factorial: параметр n < 0.
Сообщение об ошибке содержит описание причины некорректного аргумента.

4.2. std::domain_error
Выбрасывается в следующих случаях:
  - nthRoot: параметр number < 0, а параметр degree является четным числом (корень четной степени из отрицательного числа не определен в действительных числах).
  - logarithm: параметр number <= 0 (логарифм от неположительного числа не определен).
Сообщение об ошибке указывает на выход за пределы области определения функции.

4.3. std::overflow_error
Выбрасывается в следующих случаях:
  - factorial: параметр n > 20. Результат не помещается в тип unsigned long long.
Рекомендация: для вычисления факториалов при n > 20

5. ПРИМЕР ИСПОЛЬЗОВАНИЯ
Ниже приведен минимальный рабочий пример, демонстрирующий подключение модуля, использование пространства имен Algebra, вызов функций и обработку возможных исключений.

```cpp
#include <iostream>
#include <string>
#include "algebra.h"

int main() {
    // Явное использование пространства имен Algebra
    using namespace Algebra;

    try {
        // Вычисление корня кубического из 27
        double root = nthRoot(27.0, 3);
        std::cout << "Корень 3-й степени из 27: " << root << std::endl;

        // Возведение 2 в степень 10
        double pow_result = power(2.0, 10.0);
        std::cout << "2 в степени 10: " << pow_result << std::endl;

        // Логарифм 100 по основанию 10
        double log_result = logarithm(100.0, 10.0);
        std::cout << "log10(100): " << log_result << std::endl;

        // Факториал 5
        unsigned long long fact = factorial(5);
        std::cout << "5! = " << fact << std::endl;

        // Большой факториал (вывод первых 20 символов)
        std::string large_fact = factorialLarge(50);
        std::cout << "50! начинается с: " << large_fact.substr(0, 20) << "..." << std::endl;

    } catch (const std::invalid_argument& e) {
        std::cerr << "Ошибка аргумента: " << e.what() << std::endl;
    } catch (const std::domain_error& e) {
        std::cerr << "Ошибка области определения: " << e.what() << std::endl;
    } catch (const std::overflow_error& e) {
        std::cerr << "Ошибка переполнения: " << e.what() << std::endl;
    } catch (const std::exception& e) {
        std::cerr << "Неизвестная ошибка: " << e.what() << std::endl;
    }

    return 0;
}
```
